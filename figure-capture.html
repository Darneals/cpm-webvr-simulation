How to use this page:

1. Select a figure from the grid above. Each button loads a different scene configuration.

2. Click LOAD SELECTED SCENE — the 3D scene renders behind this panel.

3. Click CAPTURE FIGURE NOW — the image downloads automatically at 3900 × 2194 px (600 DPI at 6.5 inch print width).

4. For Figure 6 (Consequence Cascade): after loading, click Trigger Cascade button that appears, wait 2 seconds for the animation, then capture.

5. Rename each downloaded file immediately: fig4-hub-scene.png, fig5-node-d.png, fig6-cascade.png, fig7-node-b.png
